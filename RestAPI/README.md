# 🌐 Back-end appfilmes 🎥

## 🔁 Rotas

###  ✅ Funcionando

Listar todos
```
GET http://localhost:3000/filmes
```
Criar filme
```
POST http://localhost:3000/filmes
```
### ❌ Implementar
Obter 1
```
GET http://localhost:3000/filmes:id
```
Deletar filme
```
DELETE http://localhost:3000/filmes:id
```
